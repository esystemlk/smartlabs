## 🎭 Animation Timeline

### Page Load (0-2s):
```
0.0s: Background fade in
0.2s: Badge appears
0.3s: Heading appears
0.4s: Description appears
0.5s: Buttons appear
0.6s: Stats appear
0.7s: Stat 1 animates
0.8s: Stat 2 animates
0.9s: Stat 3 animates
1.0s: Stat 4 animates
```

### Scroll Animations:
```
Each section triggers when 20% visible
Stagger children by 0.1s
Cards animate up from 20px below
Fade from 0 to 1 opacity
```

### Continuous Animations:
```
Background rotation: 5.5s interval
Particle glow: 4s pulse loop
Icon rotations: 5s loop
Icon scales: 3s pulse loop
Floating cards: 4-6s float loop
```

## 🎯 Interactive States

### Buttons:
```
Default:  Solid background, shadow
Hover:    Gradient overlay, larger shadow
Active:   Scale down 95%
Focus:    Ring outline
```

### Cards:
```
Default:  Glass effect, subtle border
Hover:    Border glow, translate up, shadow increase
Active:   Scale 95% (mobile)
```

### Links:
```
Default:  Primary color, arrow icon
Hover:    Arrow translates right, gap increases
```

## 📱 Responsive Breakpoints

### Mobile (< 640px):
- 1 column layouts
- Full-width buttons
- Stacked stats (1-2 columns)
- Hidden floating cards
- Compact padding

### Tablet (640px - 1024px):
- 2 column layouts
- Side-by-side buttons
- 2-3 column stats
- Moderate padding

### Desktop (> 1024px):
- 3-4 column layouts
- Inline buttons
- 4 column stats
- Floating cards visible
- Maximum padding

## 🎨 Visual Effects

### Glass Morphism:
```css
background: rgba(card, 0.6)
backdrop-filter: blur(12px)
border: 1px solid rgba(border, 0.3)
```

### Gradients:
```css
Hero Badge: from-primary/20 via-accent-3/20 to-accent-1/20
CTA Button: from-primary via-accent-3 to-accent-1
Text: from-primary via-accent-3 to-accent-1
```

### Shadows:
```css
Default: shadow-lg
Hover: shadow-2xl
Glow: shadow-accent-1/50
```

---

## 🔄 Animation Patterns

### Entrance:
```
opacity: 0 → 1
y: 20 → 0
duration: 0.6s
```

### Hover:
```
scale: 1 → 1.05
translateY: 0 → -8px
duration: 0.3s
```

### Continuous:
```
Float: translateY(0 → -10 → 0)
Pulse: scale(1 → 1.05 → 1)
Rotate: rotate(0 → 5 → -5 → 0)
```
